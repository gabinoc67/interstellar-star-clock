Formal equation panel

Compact equations for the simulator logic.

Reference and biological timing
T_ref = sin(phi_ref) T_bio = sin(phi_bio) phi_ref(t+dt) = phi_ref + dt * (2*pi*f_ref)
CST mismatch
DeltaT_CST = w_CST * |T_bio - T_ref|
Environmental forcing
F_env(t) = A_env * sin(2*pi*f_env*t) + Pulse(t)
Biological phase update
phi_bio(t+dt) = phi_bio + dt * [ 2*pi*f_bio + K * sin(phi_ref - phi_bio) + P * F_env * sin(phi_bio) * 0.25 - w_CST * (T_bio - T_ref) * 0.18 ]
Adaptive response and complexity
Adaptive ~ |F_env| * (0.55 + P) * (0.4 + DeltaT_CST) Complexity(t+dt) ~ Complexity + gain * f(Adaptive, DeltaT_CST)
Transition estimate
TransitionIntensity ~ 0.45*DeltaT_CST + 0.25*|F_env| + 0.20*Adaptive + 0.10*Complexity
