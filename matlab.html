What This Engine Is For
This simulator is a visual CST-inspired engine demonstration. It is for showing how a control system could coordinate a reactor source stage, a fusion gain stage, a synthetic mass-to-energy conversion stage, warp coils, and twin nacelles inside one synchronized display. In this corrected version, the engine can truly run, truly freeze, and fully reset. The formulas shown are demo response equations for visualization and control behavior only, not experimentally validated propulsion physics.
How It Works
Warp Factor changes field intensity, CST Phase φ changes oscillation state, and m ↔ E Converter changes how strongly the demo shifts toward field-style output. When Start is pressed, clocks advance, nacelles oscillate, the phase auto-advances, and exhaust particles spawn. When Stop is pressed, all engine state freezes. Reset returns the entire simulator to its default state.
Model Note
This is a corrected demo model. The physics are still illustrative, but now the simulator behavior is logically correct: the starfield is fixed, Stop freezes the live state, and Reset restores the whole system cleanly.
