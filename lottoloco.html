It is a verifier for historical performance testing, not proof that future lottery outcomes can be predicted.

How This Verifier Works

1) Load official modern-format Lotto Texas history.
2) Freeze one ruleset before any test.
3) For each historical test case, train only on earlier draws.
4) Generate 10 tickets for the next held-out draw.
5) Record ticket hits: 3/6, 4/6, 5/6, 6/6 and best hit per case.
6) Run random-baseline simulations with the same number of tests and tickets.
7) Estimate an empirical p-value from Monte Carlo results.
8) Use the same frozen ruleset to generate the current 10 drawings.
This is a historical verifier. A low p-value would only suggest the ruleset beat the chosen random baseline on the tested historical sample. It does not guarantee future predictive power.
