How It Works

1) Click the one button.
2) The page downloads official Lotto Texas history.
3) It parses only modern 6-number rows safely.
4) It freezes one ruleset before backtesting.
5) It tests historical held-out draws against 10 generated tickets per case.
6) It compares against random ticket baselines.
7) It displays the verifier scorecard.
8) It generates the current 10 drawing numbers.
If a browser blocks the official fetch because of local security or CORS rules, the page also tries fallback mirrors and then local cache.
