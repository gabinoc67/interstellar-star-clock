Program Purpose

This program is a live educational simulation of a CST-guided warp corridor trip. It compares the outside Earth/CST clock to the inside ship proper clock, shows how path-shortening changes Earth-frame travel time, and shows how relativistic time dilation changes ship aging versus Earth aging. It does not claim physical proof of real warp travel; it is a browser-based numerical model for testing ideas, timing relationships, and mission display behavior.

Calculation Notes

Earth-frame travel time uses shortened distance L′ = L / warp. Ship proper time uses Tship = Tearth / γ with γ = 1 / √(1 − β²). Apparent Earth-view speed uses vapp = warp × vlocal. Stabilization OFF adds only a tiny illustrative display offset; it does not replace the SR proper-time calculation.
