Orbital Radar Display

The center is Earth. Satellites are shown on orbit rings by class: low Earth orbit, medium Earth orbit, geostationary orbit, and higher orbits. This is a live orbital display, not a geographic world map.
Live fields update from TLE propagation when available If live fetch fails, cache is used If cache also fails, fallback mode keeps the display running

Meaning and Purpose

GOES = Geostationary Operational Environmental Satellite
Weather and storm monitoring.
GPS = Global Positioning System
Position, navigation, and timing.
AEHF = Advanced Extremely High Frequency
Protected military communications.
MUOS = Mobile User Objective System
Mobile military UHF communications.
SBIRS = Space Based Infrared System
Missile warning and infrared detection.
WGS = Wideband Global SATCOM
Military wideband communications.
Starlink / OneWeb / Iridium
Commercial communications and broadband.
Landsat / Sentinel / NOAA
Earth observation, climate, land, water, and weather.
