What this simulation is doing

This is a toy corridor ascent from Point A to Point B in East–Up (x–y). It blends Earth rotation, q compression, drag/lift, a simple heat proxy + coolant, and a 5-stage burn.
  • Point A (lat) changes the rotation bonus v_rot.
  • Point B (vector target) stabilizes the corridor versus “straight up”.
  • Curvature control helps keep q/heat manageable and can reduce wasted fuel (toy framing).
  • CST + UTC clocks are displayed for timing windows (no spacetime proof claim).
Atmosphere lines: 10/30/60/100/150/200 km (bold white). Target B dashed green. Stage boundaries are marked on the path.
