What This Program Is About

This is a concept simulator for an internal cone engine idea. The model combines: base thin-film tile thrust, CST synchronization, resonance gain, cone efficiency, and reaction coupling to estimate a live effective thrust output. It is a visualization and control demo, not a validated physics solver.
Program purpose:

It helps you see how a small set of engine-control assumptions might change thrust, direction, and visible engine behavior in real time. The main job of the program is to let you move sliders and instantly watch the meter and engine react.
What was removed:

Preset-roadmap text overload, repeated interpretation boxes, extra construction panels, extra bubble summaries, and secondary display areas that were not necessary for core function.
Display logic now:

Four equal independent panels. Each panel can stand on its own and aligns evenly across the screen.
