What this program shows

Overview. This visualization shows a set of coupled “strings” that vibrate continuously and exchange momentum. The driver is CST (Cosmic Standard Time): a cosmic clock that maps elapsed time since a fixed epoch into the base harmonic that excites the strings.

Modes.

  • Normal: balanced behavior close to the original program.
  • Harmonic: smoother standing-wave style motion.
  • Coupled: stronger exchange between strings.
  • Turbulent: noisier, more unstable-looking flux transport.
  • CST Lock: tighter synchronized motion with lower drift feel.

How to read the screen.

  • Blue strings: displacement.
  • Green/Pink streaks: local momentum flux direction and strength.
  • Start / Stop / Reset: run, pause, or return to fresh initial conditions.
