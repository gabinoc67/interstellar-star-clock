Explanation

This page is not labeled as a verified prediction engine yet.

It is a verification lab built to test whether the model earns that label over time. The model rules are frozen before each backtest run, then evaluated on out-of-sample draws only. The dashboard compares the model against random baselines, measures hit rates, estimates empirical significance, records an audit trail, and keeps the current-pick engine separate from the backtest logic.

A model should only be called verified after it shows repeated out-of-sample performance over many past draws, with declared rules, measurable improvement over baselines, and stable results across time windows.
