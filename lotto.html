Lotto Texas Forecast Verification Demo

Loads official-history CSV when possible, accepts downloaded CSV files, runs a simple frozen-rule backtest, compares against random picks, and generates a current forecast set.

How this demo works

This demo is a forecast and verification sandbox, not a guaranteed winning system.

It uses past Lotto Texas results to build number weights from simple fixed rules: lower-frequency numbers can receive a boost, numbers from the most recent draw can receive a penalty, optional lock numbers can be forced upward, and optional pair boosts can strengthen chosen pairs.

When you press Run Test + Forecast, the page automatically freezes the current rules. That means the settings are locked for that test run before any results are measured. This helps keep the backtest honest instead of changing the rules after seeing the outcome.

Disclaimer

This program is for educational and experimental use only. Lottery drawings are random events, and this page does not prove predictive power. Even if a backtest looks better than random in one sample, that does not guarantee future success. Use it as a statistics and pattern-testing demo, not as financial advice or a promise of winnings.
