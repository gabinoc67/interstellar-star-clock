CST Warp-Tunnel Horizon Calculator

This version tracks Earth reference time, ship proper time, CST drift, sync quality, and arrival locking. It is still a concept simulator, but the timing path now behaves more like a usable CST mission planner, including a working departure time = now control and a visible departure field.

Field equations used in this program
Distance dynamics
dD/dt = H·D − v_eff
Reachability condition
v_eff > H·D₀
Travel time from given speed
T = (1/H)·ln[v_eff / (v_eff − H·D₀)]
Required speed from given time
v_eff = (H·D₀) / (1 − e^(−H·T))
Ship proper time
T_ship = T_earth · shipRate
CST drift
ΔT = T_ship − T_earth
CST factor
F_cst = clamp(1 + coupling·(align−1) − penalty)
Adjusted speed
v_eff,CST = v_eff · F_cst
Arrival lock check
|ΔT| ≤ δt
The cosmology part is a simplified expanding-distance model. The CST part is a sandbox timing-control layer, not a validated physical theory solver.

Earth time is the reference frame. Ship time runs at the chosen ship rate. CST synthetic time is a control-layer blend between the two. If arrival locking is enabled, the simulator reduces the ship-vs-Earth timing mismatch near arrival.

How This Program Works — Step by Step

1. Enter a target distance in billions of light-years or choose a preset target.
2. Enter the cosmology value H₀. The code converts it from km/s/Mpc into yr⁻¹.
3. Choose either: speed mode to solve for travel time, or time mode to solve for required effective warp speed.
4. Set the mission departure time. Press Set departure = now to stamp the field with your current local time.
5. Enter CST controls: alignment, ship time rate, lock threshold, and coupling strength.
6. The program computes recession speed first: v_rec = H·D₀. If your effective speed is not greater than that, the target is marked unreachable.
7. It then solves the Earth-frame travel time or required speed using the simplified expanding-distance equation.
8. Ship proper travel time is computed from the ship rate: T_ship = T_earth · shipRate.
9. CST drift is measured as ΔT = T_ship − T_earth. The more drift there is, the more penalty the CST layer adds.
10. A CST factor adjusts the effective warp speed: v_eff,CST = v_eff · F_cst.
11. If arrival locking is on, the program gently pulls the ship timing closer to Earth timing near arrival until the drift is within threshold when possible.
12. The results panel shows Earth arrival, ship arrival, CST synthetic time, sync score, reachability, and lock status.
