How This Program Works

1. The live clock starts immediately when the page opens.
2. Earth reference time uses the real local time.
3. Ship time is calculated from the selected ship rate.
4. CST synthetic time blends Earth time and ship time using the CST alignment factor.
5. After pressing Calculate CST Mission, the full travel-time mission model activates.
6. The synchronization panel continues updating every second.

This is a conceptual simulator, not a validated propulsion or cosmology solver.
