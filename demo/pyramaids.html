This remains a simplified thought-experiment structural sandbox, but it is now corrected so the intended 2-ton swivel lift case starts from a realistic-looking GO setup.

What was corrected

  • Starting block weight changed from the old merged 80-ton default to a 2-ton working default.
  • Front and back arm defaults corrected for a smaller swivel-lift case.
  • GO / NO-GO logic kept strict enough to fail bad setups, but no longer starts from an impossible case.
  • Top buttons and independent panel layout remain preserved.
