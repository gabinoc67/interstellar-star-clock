CST Transit V2 — Integrated Orbit / Propulsion / Theory Layer

This version replaces scalar trip progress with integrated heliocentric motion in 2D space using position and velocity state variables. Propulsion is modeled from available electrical power, exhaust velocity, thrust, and propellant flow. CST / warp / GR terms remain theory-only overlays, clearly separated from the more physical orbital-propulsion core.

Mission Controls

The core travel model uses real state integration in 2D: (x, y, vx, vy). The craft starts near Earth and tries to intercept Mars. The CST / GR layer is optional and labeled as theory-only.
