What Changed vs the Old Version

Robot control: commands are now structured and can go to a simulator or WebSocket bridge.

Ethics: no longer mostly hand-built yes/no noise. It first applies hard safety rules, then weighs benefit, harm, reversibility, and owner priority.

Dyson memory: replaced symbolic memory rhetoric with a reliability-style preservation model using redundancy and error correction logic.

Important Limits

This is still not a real moral agent, not a true quantum computer, and not a real astrophysical Dyson shell. It is a more disciplined demo that avoids pretending random heuristics are deeper than they are.
