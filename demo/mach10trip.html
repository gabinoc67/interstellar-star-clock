Explanation • Equations • Tunables • Extra Panels
Bottom panel
This version is arranged so the buttons and fields update correctly. It is still a demo, but the internal logic is consistent: a planned Mach/altitude profile is tracked by lagged actual values, then atmosphere, pressure, heating, coolant, and stability are computed from those live states.
Equations used 1) Step interpolation: s = clamp((t - t0)/(t1 - t0), 0..1) Mach_plan = Mach0 + s·(Mach1 - Mach0) Alt_plan = Alt0 + s·(Alt1 - Alt0) 2) Response lag: Mach_actual += (Mach_plan - Mach_actual) · (1 - exp(-dt/τMach)) Alt_actual += (Alt_plan - Alt_actual ) · (1 - exp(-dt/τAlt)) 3) Atmosphere interpolation: Tamb(Alt), rho(Alt), a(Alt) 4) Speed: V = Mach_actual · a(Alt_actual) 5) Dynamic pressure: q = ½ρV² 6) Heating proxy: H = q_kPa / q_ref_kPa 7) Cold index: Cold = clamp((T_warm - Tamb)/(T_warm - T_cold), 0..1) 8) Coolant flow: mDot_cmd = clamp(m_min + gH·H + gCold·Cold + gMach·(Mach/10), m_min..m_max) 9) Skin temperature ODE: Tskin' = k_heat·H·((Tamb+Δhot)-Tskin) - k_cool_eff·(Tskin-Tamb) 10) Compression: C = 1 + (q_kPa/q_comp_kPa) + 0.12·Mach_actual 11) Stability: score = 1 - penalties(loads + heating + ramp + lag + band + inertia)
