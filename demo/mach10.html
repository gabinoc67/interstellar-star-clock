Program Notes

Simulation only
What this version now does better
  • It no longer forces the aircraft to stay green all the time.
  • Instability can now be triggered by high q∞, poor thrust margin, bad inlet effectiveness, thermal overload, and disturbances.
  • The trip continues even when unstable; this matches your request not to stop the mission.
  • The autopilot attempts to recover by reducing demanded Mach, softening the flight profile, and pushing cooling / bleed corrections.
  • If the situation cannot be fully fixed, the aircraft still moves forward in degraded operation rather than ending the trip.
Important note
This is still a browser simulation, not a certified hypersonic solver. It is much more honest than the original “always stable” version, but it still uses simplified atmosphere, inlet, and thermal proxy models.
