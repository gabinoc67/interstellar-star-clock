Mission Inputs

Reference Equations

Photon energy: E = h*c/lambda
Photon momentum: p = E/c = h/lambda
Photon flux: Phi = I/E
Radiation pressure: P = (1 + R)*I/c
Force on sail: F = P*A
Acceleration: a = F/m
Velocity gain: Delta v = a*t
Live irradiance proxy: I_live(t) = I0*f_env*(1 + d_day + d_solar + d_vib)
CST timing mismatch: DeltaT = T_vehicle - T_reference
CST correction: d(DeltaT)/dt = -K*DeltaT + forcing(t)
Penalty: J = w1*(DeltaT)^2 + w2*(dDeltaT/dt)^2 + w3*(DeltaI_mismatch)^2
