Cash 5 Automatic Point-Set Analyzer

Analyze up to 1,000 historical machine-order draws. Every number receives a separate score in each of the five columns. Numbers with the same rounded point value form an independent set for that column. The 40-ticket generator then rotates through those point sets without repeating a number inside the same ticket.

Step 1 — Enter Historical Machine-Order Draws

Paste one draw per line. The first number is Column 1, the second number is Column 2, and so forth. The newest draw should be placed at the top. The program accepts spaces, commas, tabs, or hyphens.

Automatic score: each number-column pair is scored from its full-history frequency, recent frequency, short-window activity, and current absence gap. The score is used to organize numbers into sets; it does not guarantee future lottery results.
Paste historical draws and select “Analyze Draws.”

Step 3 — Automatic Points for Every Number and Column

A number is evaluated independently in each column. For example, Number 17 may receive 3.00 in Column 1 and 2.80 in Column 5. Both placements remain available to the generator.

Step 4 — Number Sets by Point and Column

These sets are created automatically. Each column remains independent, even when the same number belongs to different point sets in other columns.

Step 5 — Generate 40 Tickets from the Column Sets

Higher-point sets receive more selection pressure, but lower-point sets are not ignored. Each set maintains its own rotating number cursor.
