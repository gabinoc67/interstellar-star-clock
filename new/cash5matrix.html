Cash 5 Independent Point-Group Matrix

Every number-column pair remains independent. The generator follows each column’s point pattern, rotates through every number in that point group, and prevents the same number from appearing twice in one row.

4. Independent Point Groups by Column

Each column has its own number sets. The same point value can contain different numbers in different columns.

A repeated point value continues from the next number in that point group. It does not restart at the first number every time.

6. Generate the 40-Row Matrix

The program selects one number from each independent column group. A number can appear again on another row and in another column, but it cannot appear twice within the same row.

Row rule: Start each new row with all 35 numbers available. After Column 1 selects a number, that number is blocked only from Columns 2–5 of the same row. The pool resets for the next row.
Build the point groups, enter all five patterns, and generate the matrix.
