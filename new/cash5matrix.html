Cash 5 Independent Machine-Column Matrix

Stage 1 analyzes previous Cash Five draws exactly in the order the machine produced them. Each position is independent: Position 1 is analyzed only against Position 1, Position 2 against Position 2, and so forth.

1. Paste Previous Machine-Order Draws

Enter one draw per line. Each valid line must contain five unique numbers from 1 through 35. You may paste plain numbers or rows that include a date.

Accepted formats

  • 3 - 4 - 16 - 17 - 19
  • 3, 4, 16, 17, 19
  • 3 4 16 17 19
  • Date followed by five numbers
  • Excel rows pasted into the panel

Machine-order rule

The program does not sort each draw before analysis. The first pasted number remains in Column 1, the second remains in Column 2, and so on.

Draw: 31 - 5 - 22 - 8 - 34

Column 1 = 31
Column 2 = 5
Column 3 = 22
Column 4 = 8
Column 5 = 34

Maximum accepted valid draws: 1,000.

3. Top, Middle and Least Numbers by Machine Column

Every column has its own independent ranking. A number may be Top in one machine position and Least in another position.

Top 5: ranks 1–5 Middle 5: ranks 16–20 Least 5: ranks 31–35
Ties are resolved consistently by number order. When two numbers have the same frequency, the lower number is ranked first.
