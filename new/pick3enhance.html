Algorithm Settings

Best play recommendation: Exact Order is the highest-payout but hardest-hit play. Any Order is better for coverage on non-triple numbers because the digits can land in any order. Exact/Any is a balanced option when the number has a strong ordered position pattern but you still want some backup coverage. Fireball is not required by this generator, but it can be added manually when you want extra replacement-digit coverage.
