Cash 5 Best 40 Outcomes Engine - NDV + DCI

Analyze Data First • Build 20 Pool • Generate Candidates • Score Combinations • Apply DCI Balance • Apply Number Caps • Select Best 40

1. Last 100 Draws + Prize Code

Paste one draw per line. Prize code: 0 = no prize, 1 = one winner, 2 = two winners.

2. NDV + DCI Analyzer

NDV scores anchor/drift numbers. DCI scores ticket shape: compressed, hybrid, or spread.

3. Prize Pair / Cluster Analyzer

Prize code 1 adds support. Prize code 2 adds double support. Prize code 0 subtracts support.

4. Pretest Daily Draws

Format: Date, Test 1: five numbers, Test 2: five numbers, Test 3: five numbers, Test 4: five numbers.

5. Today Draw

Today numbers cool down unless supported by NDV, DCI, pretest, gap, prize pair, or prize cluster logic.

6. Run Full Engine

Runs all steps in order before creating the best 40 outcomes.

7. Tomorrow 20-Number Pool

Candidate buy pool. This is created before candidate combinations are generated.

8. Build Best 40 Outcomes

No fixed 20/20 split. Scores many candidates, applies DCI shape balance and number caps, then selects the best 40.

9. Best 40 Logic Panel

Shows selection method, number caps, and DCI distribution.

10. Backtest

Tests the 20-number pool against the last 100 draws.

11. Standard Pair Engine

Shows ordinary adjacent-pair activity from the draw history.

12. Replacement Panel

Shows cooling, replacement numbers, anchors, drift numbers, and prize-supported numbers.

13. Draw Result Tracker

After official draw, compare against generated best 40 outcomes.

14. CSV / Excel Records

Exports full report including NDV, DCI, candidate scores, caps, and best 40 outcomes.

Best 40 Formula Logic

Step 1: Analyze all data first.
Step 2: Build 20-number pool.
Step 3: Generate candidate combinations.
Step 4: Score every combination.
Step 5: Apply DCI shape balance.
Step 6: Apply number-frequency caps.
Step 7: Select best 40 outcomes.

NDV = number behavior.
DCI = ticket shape behavior.
