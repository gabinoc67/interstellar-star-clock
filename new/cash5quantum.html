Cash 5 Adaptive Prize Statistics Engine

Last 100 Draws • Prize Code 0 / 1 / 2 • Winning-Number vs No-Prize Comparison • 40 Adaptive Outcomes

1. Last 100 Draws + Prize Code

Paste one draw per line. Add prize code at the end.
0 = no prize winner, 1 = one individual won, 2 = two individuals won.

2. Prize Winner Statistics

Analyzes which numbers appear more often when prize code is 1 or 2, and which numbers appear more often when prize code is 0.

3. A1 / A2 / A3 Pretests

Paste pretests. A1, A2, A3 are same-day machine pretest signals.

4. Today Draw

Enter today’s real draw. Example: 14 18 25 26 31.

5. Run Full Engine

Runs draw frequency, prize code analysis, pretests, today cooling, pair clusters, and 40 outcomes.

6. Tomorrow 20-Number Pool

Builds 20 numbers using draw pressure, gap pressure, pretests, pairs, and prize-winning strength.

7. Previous 40 Feedback

Paste previous generated number counts, actual draw, and winning notes.

8. Build 40 Outcomes

Program decides no-pair vs pair-mix ratio from recent data.

9. Adaptive Ratio

Shows how many outcomes are no-pair and how many use pairs.

10. Backtest

Tests tomorrow 20-number pool against the last 100 draws.

11. Pair Cluster Engine

Scans common and least-used adjacent pairs.

12. Replacement Panel

Shows cooling numbers, replacements, and prize-supported numbers.

13. Draw Result Tracker

After the official draw, compare it against your 40 generated outcomes.

14. CSV / Excel Records

Exports pressure table, prize table, 20-number pool, 40 outcomes, and saved records.

Prize Code Rule

0 = no prize winner.
1 = one individual won a prize.
2 = two individuals won a prize.

The program compares each number against prize-winning draws and no-prize draws.

Formula

Final Score = Frequency + Recent Trend + Gap Pressure + Pretest Signal + Pair Strength + Prize Strength + Feedback Boost − Cooling Penalty.
