Cash 5 Adaptive 40 Outcome Pressure Engine

Cash 5 5/35 • Data-Driven Ratio • 40 Outcomes • No-Pair + Pair-Mix Auto Split

1. Last 100 Draws

Paste official drawings. One draw per line. The engine uses last 100 valid draws.

2. A1 / A2 / A3 Pretests

Paste pretests. A1, A2, A3 are same-day machine pretest signals.

3. Today Draw

Enter today’s real draw. Example: 14 18 25 26 31.

3A. Daily Draw Result / CSV Record

Enter the official drawn numbers after the draw. The program compares them with your generated 40 tickets and saves a CSV-ready record.

4. Run Full Engine

Runs all analysis and builds 40 adaptive outcomes.

5. Tomorrow 20-Number Pool

Generates 20 numbers using pressure, hot, natural, gap, pretest, and pair strength.

6. Build 40 Outcomes

Program decides no-pair vs pair-mix ratio from your data.

7. Adaptive Ratio Panel

Shows how many outcomes will be no-pair and how many will use pairs.

8. Backtest Panel

Tests tomorrow 20-number pool against the last 100 draws.

9. Pair Cluster Engine

Scans your data for common and least-used adjacent pairs.

10. Replacement Panel

Shows cooling numbers, replacements, and pair support.

Adaptive Rule

The program analyzes your pasted draws.

If recent data has more pairs, it creates more pair-mix outcomes.

If recent data has more no-sequence draws, it creates more no-pair outcomes.

Minimum pair coverage: 20% of 40 = 8 pair outcomes.

Formula

Tomorrow Score = Long Frequency + Recent Frequency + Gap Pressure + Pretest Signal + Natural Stability + Pair Strength − Today Cooling Penalty.

Result Overlap Record = Draw Number Appearances in 40 Tickets + Hit Pair Repetition + Free-Ticket Count.
