Cash 5 Prize Pair Cluster Adaptive Engine

Cash 5 5/35 • Prize Code 0/1/2 • Prize Pair + Cluster Analyzer • Fixed 20 No-Pair + 20 Mix-Pair Outcomes

1. Last 100 Draws + Prize Code

Paste one draw per line. Add prize code at end. 0 = no prize winner, 1 = one individual won, 2 = two individuals won.

2. Prize Pair / Cluster Analyzer

Must run before 40 outcomes. It finds pairs and 3-number clusters that appear in prize code 1 or 2 draws.

3. A1 / A2 / A3 Pretests

Paste pretests. A1, A2, A3 are same-day pretest signals.

4. Today Draw

Enter today’s real draw. These receive cooling unless supported by strong signals.

5. Run Full Engine

Runs prize pair/cluster analysis first, then builds 20 no-pair and 20 mix-pair outcomes.

6. Tomorrow 20-Number Pool

Uses frequency, gap, pretest, today cooling, prize pairs, and prize clusters.

7. Previous 40 Feedback

Optional. Paste previous generated count, actual draw, and notes.

8. Build 40 Outcomes

Fixed split: 20 no-pair outcomes and 20 mix-pair outcomes.

9. Fixed Ratio Panel

Shows fixed 20 / 20 split.

10. Backtest

Tests the 20-number pool against the last 100 draws.

11. Standard Pair Engine

Shows ordinary adjacent-pair activity from the draw history.

12. Replacement Panel

Shows cooling, replacements, and prize-supported numbers.

13. Draw Result Tracker

After official draw, compare against generated 40 outcomes.

14. CSV / Excel Records

Exports full report including prize pairs and prize clusters.

Prize Formula

Prize Pair Score = Prize 1 Pair Count + 2 × Prize 2 Pair Count − Prize 0 Pair Count.

Prize Cluster Score = Prize 1 Cluster Count + 2 × Prize 2 Cluster Count − Prize 0 Cluster Count.

The engine analyzes this before creating the 40 outcomes.

Outcome Rule

The 40 outcomes are fixed:

20 no-pair / no-adjacent outcomes.

20 mix-pair outcomes using prize-supported pairs, prize clusters, ordinary pairs, pretests, and pressure numbers.

This is statistical comparison only and does not guarantee lottery results.
