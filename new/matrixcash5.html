Cash 5 Point Sequence Rotation Matrix

Paste the historical machine-order file with the oldest draw at the top and the newest draw at the bottom. The program keeps that order unchanged, reads the newest scoring windows from the bottom, and creates the newest 40 point sequences by moving upward from the newest draw.

Step 1 — Paste Historical Machine-Order Draws

Paste one draw per line exactly as it appears in the downloaded file. The oldest draw must be at the top and the newest draw at the bottom. The program does not reverse the historical file. The first number is Column 1, the second is Column 2, and so forth. Spaces, tabs, commas, semicolons, pipes, or hyphens are accepted.

Historical order:
Oldest draw at the top → newest draw at the bottom.
The program preserves this order. The newest 100, newest 25, and newest 40 are read directly from the bottom of the pasted history.
Rotation rule: each point value has its own independent number cursor in each column. The first occurrence uses the first number in that point group, the next occurrence uses the next number, and the cursor returns to the beginning after the final member. A single-number point always reuses that number.
Paste at least 40 valid historical draws with the oldest at the top and the newest at the bottom.

Step 4 — Single Numbers and Point Groups

Numbers inside each point group are ordered from lowest to highest. That displayed order is the same order used by the rotation engine.

Step 5 — Newest 40 Draws Converted to Point Sequences

Point Sequence Row 1 is the newest historical draw. The program moves upward through the final 40 historical rows without changing the stored oldest-to-newest history.
