Cash 5 Clean Formation Matrix Engine

Analyze 170–200 pasted Cash 5 drawings and generate 40 diversified candidate rows using frequency, momentum, repeat bridges, recovery cycles, cycle anchors, pairs, triples, active sums, families, and historical gap shapes.

1. Paste Drawing History

Paste the full table exactly as copied. The newest drawing should be at the top and the oldest drawing at the bottom. The parser reads the date, five winning numbers, prize amount, and number of top-prize winners. Header lines are automatically ignored.

The example drawing history is not built into this program. Analysis and generation use only the drawings pasted below.
